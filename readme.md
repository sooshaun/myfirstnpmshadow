# Installation

`npm - i - myfirstnpm --save`

Then import to the file that you would like to use this feature

import {shadowizard} from 'myfirstnpm'

`
shadowizard({
  shadow_type: 'soft' || 'hard',
  padding: 'false' || 'true'
});

'

There are two options

* *shadow_type* - _hard | soft_ (default to soft)
